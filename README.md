# AI Resume Analyzer 🚀

A premium, enterprise-grade AI Resume Analyzer built with Flask and Python. This application parses document metadata, maps professional keywords against target job metrics, and renders a real-time matching index dashboard.

## 📊 Live System Output
Paste your target job description matrix metrics, terminal logs, or score metrics right here! For example:

```text
[ANALYSIS LOG] Target Job: Backend Engineer (Data Infrastructure)
[PARSER SUCCESS] Document 'KS_Sindhuja_Resume (2).pdf' ingested.

[OPTIMIZATION INDEX] Core Match Score: 88%
